Installation

System Requrements

*** PHP 5.6 >= 7.3

*** MYSQL 5+

*** PHP Mcrypt Extention

*** PHP MBString Extension

*** PHP GD Extension

*** PHP MYSQLI Extension

*** PHP CURL Extention

*** PHP ZIP Extention

*** Enable allow_url_fopen

Installation

  1. Unzip and copy the script folder in your hosting server.
  2. Make sure in script folder index.php file have permission as 644
  3. Make sure in script /mvc/libraries, /mvc/config and /uploads folder have permission as 777
  4. Manually create a database using “Phpmyadmin”.
  5. Run the install script path from any web browser (http://server.com/index.php/install/index).
  6. If check list seems all ok then click next
  7. Provide database hostname, database name, username and password.
  8. Provide your country time zone and click next.
  9. Fill up organization information.
  10. Provide system admin information and click next.
  11. Now click 'go to login'
  12. Good Luck!!!

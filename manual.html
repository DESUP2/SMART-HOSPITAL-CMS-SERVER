Installation

Unzip and copy the script folder in your hosting server.
Make sure in script folder index.php file have permission as 644
Make sure in script /mvc/libraries, /mvc/config and /uploads folder have permission as 777
Manually create a database using “Phpmyadmin”.
Run the install script path from any web browser (http://server.com/index.php/install/index).
If check list seems all ok then click next
Provide database hostname, database name, username and password.
Provide your country time zone and click next.
Fill up organization information.
Provide system admin information and click next.
Now click 'go to login'
Good Luck!!!
